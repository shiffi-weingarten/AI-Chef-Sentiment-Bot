# AI Virtual Chef & Sentiment Analyzer 🍳🤖

פרויקט Python מתקדם המשלב בינה מלאכותית (Gemini AI) עם ניתוח נתונים (Data Analysis) ליצירת חוויית ייעוץ קולינרית אינטראקטיבית.

---

## 📝 אודות הפרויקט
המערכת מדמה שף מקצועי המלווה את המשתמש בתהליכי בישול ואפייה. הייחודיות של הפרויקט היא בשימוש באלגוריתם **ניתוח סנטימנט (Sentiment Analysis)** העוקב אחר שביעות הרצון של המשתמש לאורך השיחה ומפיק דוח ויזואלי בסיומה.



[Image of sentiment analysis process flow]


## ✨ תכונות עיקריות במערכת
* **בינה מלאכותית (LLM):** אינטגרציה מלאה עם מודל `gemini-2.5-flash` של Google.
* **ניתוח רגשות דינמי:** חישוב ציון רגש לכל הודעה על בסיס מילון מונחים מותאם אישית תוך שימוש ב-`Dictionary Comprehension`.
* **ויזואליזציה של נתונים:** הפקת גרפי קו ועמודות (Matplotlib) המציגים את מגמת הסנטימנט ואת ההבדלים בין תגובות המשתמש לשף.
* **חוסן מערכתי (Mock Mode):** מנגנון גיבוי המאפשר לבוט להמשיך לתפקד גם ללא חיבור פעיל ל-API (שימוש בנתוני דמה).
* **ניהול קבצים מתקדם:** יצירת שמות קבצים חכמים בעזרת ה-AI ושמירת היסטוריית השיחה בפורמט JSON.
* **שימוש ב-Generators:** יישום יעיל של שליפת מילים שליליות מתוך הנתונים ללא העמסה על הזיכרון.



## 🛠 טכנולוגיות וכלים
* **Python 3.13**
* **Google GenAI SDK** (Gemini API)
* **Matplotlib & Pandas** (ניתוח והצגת נתונים)
* **JSON & Regex** (עיבוד טקסט ואחסון)
